Voxel Architecture
Voxel Architecture investigates the coplexity of emergent geometric compositions following the simplest evolutionary rules.
Voxel Architecture discretizes the site to a set of voxels of different contents and assmues that a well-organization of these contents can be achieved by swapping the contents between voxels.
Voxel Architecture demonstrates this idea using Voronoi cells with different cell ratios and objective functions that controls the compactness of the geometry.
